# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project, you agree to abide by its terms.

## Reporting

If you experience or witness unacceptable behavior, report it privately
by contacting an administrator on the official
[support Discord server](https://discord.com/invite/9Xj68BpKRb).

**Do not open a public GitHub issue for conduct reports.** Keeping these
reports off a public tracker protects reporters and preserves the privacy
of everyone involved.

Reports will be reviewed and handled promptly and fairly.
